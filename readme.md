# Kollométator
## Installation
Première utilisation :
```bash
python3.10 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Fichier contenant les infos critiques : .env
```
TOKEN=xxxxxxxxxxxxxxxxxx
DEBUG=True
```

Lancement du bot :

```bash
source .venv/bin/activate
python bot.py
```
